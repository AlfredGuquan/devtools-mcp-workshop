关于 GDG 和 DevFest

Google Developer Groups (GDG)

Google Developer Groups 是一个面向开发者的全球社区网络。GDG 为开发者提供了一个平台， 让他们可以学习、分享和讨论 Google 技术，以及探索最新的开发趋势和最佳实践。

GDG 社区由志愿者组织，定期举办技术分享会、工作坊和开发者活动。 无论你是初学者还是经验丰富的开发者，GDG 都欢迎你的加入。

DevFest

DevFest 是 GDG 社区每年举办的最重要的年度活动之一。这是一个全球性的开发者节日， 旨在庆祝技术创新、知识分享和社区建设。

DevFest 活动通常包括技术演讲、实践工作坊、黑客马拉松和网络交流环节。 它为开发者提供了一个独特的机会，可以与同行交流、学习新技术，并探索最新的开发工具和框架。

本次 Workshop 作为 DevFest 的一部分，专注于 Chrome DevTools MCP Server 这一前沿技术， 为参与者提供深入的学习和实践体验。
